# Proyecto Grupal – Odontología Aquino

## Instrucciones para ejecutar el proyecto


### Pasos para ejecutar

1. Copiar la carpeta del proyecto o clonar el repositorio.

2. Abrir una terminal en la carpeta raíz del proyecto:
   ```bash
   cd proyecto_grupal 

3. Ejecutar el proyecto con Docker:
   docker-compose up --build

4. Abrir un navegador web y acceder a:
   http://localhost:5000
